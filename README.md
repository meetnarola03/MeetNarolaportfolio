# Meet Vipulbhai Narola - Portfolio Website

A modern, production-ready portfolio website built with React, Vite, and Tailwind CSS.

## 🚀 Technologies

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router
- **UI Components**: Radix UI (shadcn/ui)
- **Icons**: Lucide React
- **Deployment**: Vercel-ready

## 📋 Features

- ✅ Responsive mobile-first design
- ✅ Light/dark theme toggle with persistence
- ✅ 5 detailed project case studies
- ✅ SEO optimized with meta tags and structured data
- ✅ Accessible routing and keyboard navigation
- ✅ Contact form with email fallback
- ✅ Skills showcase with editable levels
- ✅ Smooth animations and transitions

## 🛠️ Local Development

### Prerequisites

- Node.js 16+ and npm

### Installation & Setup

```bash
# Clone the repository
git clone <YOUR_GIT_URL>
cd <PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The development server will start at `http://localhost:8080`.

## 📦 Build & Deploy

### Build

```bash
npm run build
```

The optimized production build will be in the `dist` folder.
